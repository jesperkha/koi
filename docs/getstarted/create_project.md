# Creating a project

Create a directory for you Koi project and `cd` into it:

```sh
$ mkdir my-project
$ cd my-project
```

Run `koi init` to initialize an empty Koi project with default configuration.

```sh
$ koi init
```

Result:

```
/my-project
    /src
        main.koi
    .gitignore
    koi.toml
    README.md
```

Run with `koi run`.

```sh
$ koi run
Hello, world!
```
